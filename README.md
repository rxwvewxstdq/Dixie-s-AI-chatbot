# С Дикси готовить дешевле

Готовый React/Vite сайт с разделами:

- главная страница со слоганом «С Дикси готовить дешевле»;
- главные скидки недели;
- рецепты из продуктов со скидкой;
- таблица сравнения цен с другими магазинами;
- адаптивная верстка для телефона и компьютера.

## Как запустить

```bash
npm install
npm run dev
```

После запуска откройте адрес, который покажет Vite, обычно:

```bash
http://localhost:5173
```

## Сборка

```bash
npm run build
```

Готовая сборка появится в папке `dist`.

## Важно

Цены и сравнения в проекте демонстрационные. Для реального сайта их нужно заменить на актуальные данные.
