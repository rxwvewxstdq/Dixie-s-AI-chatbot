# Dixy Print Platform

Полностью готовый к запуску Next.js-проект для магазина Дикси.

## Что умеет

- регистрация и вход
- отправка принтов и картинок
- премодерация: все новые работы сначала попадают администратору
- администратор может:
  - одобрять публикацию
  - отклонять с причиной
  - запрещать пользователю отправку новых заявок
  - снимать запрет
  - разбирать жалобы на принты
  - снимать работу с публикации по жалобам
- голосование за опубликованные идеи
- комментарии к работам
- жалобы на принты
- email-уведомления:
  - приветственное письмо после регистрации
  - подтверждение отправки идеи
  - уведомление об одобрении
  - уведомление об отклонении
  - письмо админу о новой заявке
  - письмо админу о новой жалобе
- красивая административная панель
- Prisma + SQLite

## Быстрый запуск

```bash
npm install
cp .env.example .env
npm run setup
npm run dev
```

Открыть:
- сайт: `http://localhost:3000`
- админ: `http://localhost:3000/admin`

## Админ по умолчанию

После `npm run prisma:seed` создаётся админ:

- email: `admin@dixy.local`
- пароль: `ChangeMe123!`

## Email

Если SMTP не указан, письма сохраняются в `storage/emails`.

## Production

Для production рекомендуется заменить SQLite на PostgreSQL и подключить SMTP.

npm install && cp .env.example .env && npm run setup && npm run dev
